# Team Nepal Homepage

A static website built by Team Nepal as a school project.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page |
| `discover.html` | Discover schools |
| `news.html` | News section |
| `contact.html` | Contact form and location |

## Tech Stack

- HTML5
- CSS3
- Google Fonts (Philosopher, Poppins)

## How to Run

Just open `index.html` in your browser — no setup needed.
